# Setting Up Your CyberWacht Project

This guide will walk you through the initial steps of setting up your project on the CyberWacht platform. Follow these steps to get started:

### 1. Create an Account:

1. Visit the CyberWacht website and sign up for a new account.
2. Provide the necessary information, such as your email address, password, and organization details.

### 2. Project Setup:

1. Once you've logged in, you'll be prompted to create a new project.
2. Provide a name and description for your project.
3. Choose the appropriate project type (e.g., Web Application, Mobile App, Infrastructure).

### 3. Add Your Repository:

#### Public Repository:

1. Enter the public URL of your GitHub repository.

#### Private Repository:

1. Enable the **Is this a private repository?** option.
2. Authenticate with GitHub to grant CyberWacht access to your private repository.

### 4. Add Your Domain

1. If you want to scan a specific domain, click the **Add Domain** button.
2. Enter the domain URL and an optional label.

Once you've completed these steps, CyberWacht will start analyzing your project and providing security insights.
